Allows images in the media library to be direclty download one by one or in bulk.

== Description ==

Download Image allows you to directly download images from the media library to your device in one click.

You can download images one by one, or in bulk.

= Need Help? Found A Bug? Want To Contribute Code? =

Support for this plugin is provided via the [WordPress.org forums](https://wordpress.org/support/plugin/download-image).

The source code for this plugin is available on [GitHub](https://github.com/JeanDavidDaviet/download-image).

== Installation ==

1. Go to your admin area and select Plugins → Add New from the menu.
2. Search for "Download Image".
3. Click install.
4. Click activate.
5. Navigate to Media → Library
